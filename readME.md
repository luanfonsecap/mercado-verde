# Mercado Verde
Comer bem, faz bem !

## 💪️ O Projeto

Mercado Verde é uma plataforma de marketplace de produtos orgânicos com o foco em conectar pequenos produtores rurais ao cliente final.

## ⚗️ Tecnologias utilizadas
	- Node.js
	- Express
	- Sequelize
	- Bcrypt
